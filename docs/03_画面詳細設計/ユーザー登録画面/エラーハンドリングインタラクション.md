# エラーハンドリングインタラクション

## インタラクション概要

ユーザー登録画面で発生する各種エラーの処理と、ユーザーへの適切なフィードバック表示に関する動作を定義します。

## エラー分類とインタラクション

### 1. 入力バリデーションエラー

#### 必須項目未入力
```
[未入力項目検出処理]
    ├─ フォーム送信時一括チェック
    ├─ 項目別チェック実行
    ├─ エラー項目リスト作成
    ├─ 最初のエラー項目特定
    └─ エラーサマリー生成

[未入力エラー表示]
    ├─ 項目枠線：赤色変更
    ├─ 必須マーク強調表示
    ├─ "この項目は必須です"メッセージ
    ├─ フォーカス自動移動
    └─ エラーカウンター更新

[エラー解消処理]
    ├─ 入力開始検知
    ├─ エラー状態解除
    ├─ 成功状態表示
    ├─ 次項目への誘導
    └─ 全体進捗更新
```

#### 形式不正エラー
```
[形式エラーパターン]
    ├─ メールアドレス形式不正
    ├─ パスワード強度不足
    ├─ 氏名文字種不正
    ├─ ふりがな形式不正
    └─ 生年月日論理エラー

[メールアドレス形式エラー]
    ├─ "@マークがありません"
    ├─ "ドメイン名が不正です"
    ├─ "メールアドレスが長すぎます"
    ├─ "使用できない文字が含まれています"
    └─ "正しい形式で入力してください（例：user@example.com）"

[パスワード強度エラー]
    ├─ "8文字以上で入力してください"
    ├─ "英字と数字を含めてください"
    ├─ "大文字と小文字を含めてください"
    ├─ "記号を1文字以上含めてください"
    └─ "一般的すぎるパスワードです"
```

#### 業務ロジックエラー
```
[重複チェックエラー]
    ├─ メールアドレス重複
    ├─ ユーザー名重複
    ├─ 電話番号重複
    └─ その他識別子重複

[重複エラー表示]
    ├─ "このメールアドレスは既に登録されています"
    ├─ 代替案提示："別のメールアドレスをお試しください"
    ├─ ログイン案内："既にアカウントをお持ちですか？"
    ├─ パスワードリセット案内
    └─ サポート連絡先表示

[年齢制限エラー]
    ├─ 最低年齢未満検出
    ├─ "13歳以上の方のみご利用いただけます"
    ├─ 保護者同意案内
    ├─ 利用規約説明
    └─ 代替サービス案内
```

### 2. サーバー側エラー

#### APIエラーレスポンス
```
[HTTPステータス別処理]
    ├─ 400 Bad Request → リクエスト形式エラー
    ├─ 409 Conflict → データ重複エラー
    ├─ 422 Unprocessable Entity → バリデーションエラー
    ├─ 429 Too Many Requests → レート制限エラー
    └─ 500 Internal Server Error → サーバーエラー

[400 Bad Requestエラー]
    ├─ "送信データに問題があります"
    ├─ エラー詳細の表示
    ├─ 修正必要項目の特定
    ├─ フォーム再送信案内
    └─ 技術サポート連絡先

[429 Rate Limitエラー]
    ├─ "一時的に利用制限されています"
    ├─ 制限解除時刻表示
    ├─ 残り時間カウントダウン
    ├─ フォーム一時無効化
    └─ 制限理由の説明
```

#### データ保存エラー
```
[データベースエラー処理]
    ├─ 接続エラー検出
    ├─ タイムアウトエラー処理
    ├─ 整合性制約違反
    ├─ ストレージ容量不足
    └─ トランザクション失敗

[保存失敗時の対応]
    ├─ "登録処理中にエラーが発生しました"
    ├─ 入力データの自動保存
    ├─ 再試行ボタン提供
    ├─ 一時保存機能案内
    └─ オフライン登録案内

[自動復旧処理]
    ├─ 自動リトライ機能（最大3回）
    ├─ 指数バックオフ適用
    ├─ 部分的データ復旧
    ├─ 整合性チェック実行
    └─ ユーザー通知機能
```

### 3. ネットワークエラー

#### 接続エラー
```
[ネットワーク状態検知]
    ├─ オンライン/オフライン判定
    ├─ 接続品質測定
    ├─ DNS解決確認
    ├─ プロキシ設定確認
    └─ ファイアウォール確認

[オフライン状態処理]
    ├─ "インターネット接続を確認してください"
    ├─ オフラインモード案内
    ├─ データローカル保存
    ├─ 接続復旧時自動同期
    └─ 手動同期ボタン提供

[接続復旧時処理]
    ├─ 自動接続確認
    ├─ 保存データ同期
    ├─ エラー状態解除
    ├─ 正常状態復帰
    └─ 成功メッセージ表示
```

#### タイムアウトエラー
```
[タイムアウト種別]
    ├─ 接続タイムアウト（10秒）
    ├─ 読み取りタイムアウト（30秒）
    ├─ バリデーションタイムアウト（5秒）
    └─ ファイルアップロードタイムアウト（60秒）

[タイムアウト処理]
    ├─ "通信がタイムアウトしました"
    ├─ 現在の処理状況説明
    ├─ 再試行オプション提供
    ├─ より短い処理への分割提案
    └─ オフライン保存案内

[リトライ機能]
    ├─ 手動リトライボタン
    ├─ 自動リトライオプション
    ├─ 進捗状況表示
    ├─ キャンセル機能
    └─ タイムアウト時間調整
```

### 4. セキュリティエラー

#### CSRF/セッションエラー
```
[セキュリティトークンエラー]
    ├─ CSRFトークン不正
    ├─ セッション期限切れ
    ├─ トークン改ざん検出
    ├─ 不正なリクエスト検出
    └─ セキュリティ違反検出

[セキュリティエラー対応]
    ├─ "セキュリティ確認が必要です"
    ├─ ページ再読み込み案内
    ├─ 再ログイン要求
    ├─ セキュリティ説明表示
    └─ サポート連絡先案内

[自動回復処理]
    ├─ 新しいトークン取得
    ├─ セッション再確立
    ├─ フォームデータ復元
    ├─ 自動再送信
    └─ セキュリティログ記録
```

#### アカウント制限エラー
```
[制限パターン]
    ├─ IP制限
    ├─ 地域制限
    ├─ デバイス制限
    ├─ 同時接続制限
    └─ 利用時間制限

[制限エラー表示]
    ├─ "現在このサービスはご利用いただけません"
    ├─ 制限理由の説明
    ├─ 制限解除条件の案内
    ├─ 代替手段の提案
    └─ サポート連絡先情報
```

### 5. ファイルアップロードエラー

#### プロフィール画像エラー
```
[ファイル形式エラー]
    ├─ 対応形式チェック（JPEG, PNG, GIF）
    ├─ ファイルサイズ制限（5MB）
    ├─ 画像解像度確認
    ├─ ファイル破損チェック
    └─ ウイルススキャン

[アップロードエラー処理]
    ├─ "対応していないファイル形式です"
    ├─ "ファイルサイズが大きすぎます"
    ├─ "画像ファイルを選択してください"
    ├─ リサイズ機能提案
    └─ ファイル選択やり直し案内

[アップロード進捗表示]
    ├─ プログレスバー表示
    ├─ 処理状況テキスト
    ├─ 推定残り時間
    ├─ キャンセルボタン
    └─ 成功/失敗フィードバック
```

## エラー表示デザインパターン

### 統合エラーサマリー
```
[画面上部エラーバナー]
    ├─ エラー総数表示
    ├─ "○件のエラーがあります"
    ├─ エラー項目一覧
    ├─ 各項目へのジャンプリンク
    └─ 一括修正支援機能

[エラーバナーの動作]
    ├─ エラー発生時：スライドダウン表示
    ├─ エラー解消時：自動更新
    ├─ 全解消時：フェードアウト
    ├─ 閉じるボタン：手動非表示
    └─ 再表示機能：エラー再発時
```

### フィールド別エラー表示
```
[インラインエラー]
    ├─ フィールド直下配置
    ├─ エラーアイコン + メッセージ
    ├─ 赤色枠線 + 背景色変更
    ├─ アニメーション：スライドダウン
    └─ フォーカス時の強調表示

[エラー状態の継承]
    ├─ フォーム送信時まで保持
    ├─ 修正開始時に解除
    ├─ 部分的修正の反映
    ├─ 関連項目の連動更新
    └─ 全体状態との同期
```

### モーダル型重要エラー
```
[重大エラー用モーダル]
    ├─ セキュリティエラー
    ├─ システムメンテナンス
    ├─ 利用制限通知
    ├─ アカウント停止通知
    └─ 緊急メンテナンス案内

[モーダル構成]
    ├─ 背景オーバーレイ
    ├─ 中央配置ダイアログ
    ├─ エラーアイコン（警告色）
    ├─ エラータイトル
    ├─ 詳細説明
    ├─ 対処法案内
    └─ アクションボタン
```
