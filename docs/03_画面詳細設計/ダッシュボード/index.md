# ダッシュボード詳細設計

## 画面概要

ユーザーがログイン後に最初に表示されるメイン画面です。家計簿の概要情報、最近の支出、予算状況を一覧できます。リアルタイムで支出状況を把握し、予算管理をサポートします。
