# ダッシュボード画面表示インタラクション

## インタラクション概要

ダッシュボード画面の初期表示から各種データ表示、ユーザー操作に対する動的な画面更新までの一連の動作を定義します。

## インタラクションフロー

### 1. 初期表示シーケンス
```
[ダッシュボード画面アクセス]
    ↓
[認証状態確認]
    ├─ 未認証 → ログイン画面へリダイレクト
    └─ 認証済み → 初期データ読み込み開始
    ↓
[スケルトン画面表示]
    ├─ サマリーカード: スケルトンローダー
    ├─ グラフエリア: プレースホルダー
    └─ リストエリア: スケルトン行表示
    ↓
[データ取得・段階表示]
    ├─ 基本データ取得 → サマリーカード更新
    ├─ グラフデータ取得 → チャート描画
    ├─ 履歴データ取得 → 最近の支出リスト更新
    └─ 分析データ取得 → 目標・アドバイス表示
```

### 2. サマリーカードインタラクション

#### 今月支出カード
```
[今月支出カードクリック]
    ↓
[詳細モーダル表示]
    ├─ 日別支出グラフ
    ├─ 週別比較
    ├─ カテゴリ別内訳
    └─ [詳細分析へ]ボタン → レポート画面遷移

[前月比インジケーター]
    ├─ 増加時: 📈 赤色アイコン + 上昇アニメーション
    ├─ 減少時: 📉 緑色アイコン + 下降アニメーション
    └─ 同じ時: ➡️ グレーアイコン + 横移動

[数値更新アニメーション]
    ├─ カウンターアニメーション（1.5s）
    ├─ 桁区切りカンマの動的表示
    └─ 色変化エフェクト（予算状況に応じて）
```

#### 予算達成率カード
```
[予算達成率プログレスバー]
    ├─ 0-70%: 緑色 + 安全メッセージ
    ├─ 71-90%: 黄色 + 注意メッセージ
    ├─ 91-100%: 橙色 + 警告メッセージ
    └─ 100%超過: 赤色 + 緊急メッセージ + 点滅効果

[プログレスバーアニメーション]
    ├─ 初期表示: 0% → 実際の値まで1.5s
    ├─ 更新時: 現在値 → 新値まで1.0s
    └─ ホバー時: 詳細ツールチップ表示

[カードタップ（モバイル）]
    ↓
[予算設定画面へ遷移] + フェードアウト効果
```

#### 残り予算カード
```
[残り予算表示ロジック]
    ├─ 正の値: 緑色 + "余裕があります"
    ├─ 少額（予算の10%以下）: 黄色 + "残りわずかです"
    ├─ 負の値: 赤色 + "予算を超過しています"
    └─ 1日平均計算: (残り予算 ÷ 残り日数)

[カードホバー効果]
    ├─ 影の拡大
    ├─ 軽微な上昇アニメーション
    └─ 詳細情報のオーバーレイ表示
```

### 3. グラフインタラクション

#### 月次支出推移グラフ
```
[グラフ初期描画]
    ↓
[データポイント順次表示]
    ├─ 左から右へ順次描画（0.5s間隔）
    ├─ 各棒の高さアニメーション
    └─ 完了時に平均線描画

[棒グラフ要素ホバー]
    ↓

    ├─ 月・金額・予算比較
    ├─ 前月比の増減
    ├─ カテゴリ別詳細リンク
    └─ 位置調整（画面端対応）

[棒グラフクリック]
    ↓
[該当月詳細表示]
    ├─ モーダルウィンドウ表示
    ├─ 日別支出カレンダー
    ├─ カテゴリ別円グラフ
    └─ [その月の履歴を見る]ボタン
```

#### カテゴリ別円グラフ
```
[円グラフ描画アニメーション]
    ├─ 回転描画（時計回り）
    ├─ セクション順次表示
    ├─ ラベル・パーセンテージ表示
    └─ レジェンド同期表示

[セクションホバー]
    ├─ 該当セクション拡大（15%）
    ├─ 他セクション半透明化
    ├─ 詳細情報ツールチップ
    └─ レジェンドハイライト

[セクションクリック]
    ↓
[カテゴリ詳細ドリルダウン]
    ├─ 該当カテゴリの支出履歴表示
    ├─ 日別支出推移グラフ
    ├─ 予算比較・達成率
    └─ [履歴画面で詳細を見る]

[レジェンドインタラクション]
    ├─ カテゴリ名クリック → 表示切替
    ├─ 色アイコンクリック → セクション強調
    └─ ダブルクリック → 単独表示
```

### 4. 最近の支出リストインタラクション

#### リスト表示・更新
```
[支出リスト初期表示]
    ↓
[アイテム順次表示]
    ├─ 上から下へフェードイン（0.2s間隔）
    ├─ 時刻・カテゴリアイコン・金額表示
    └─ スワイプアクション準備

[リアルタイム更新]
    ├─ 新規支出追加時: 最上部に挿入 + ハイライト
    ├─ 支出編集時: 該当行更新 + 点滅効果
    ├─ 支出削除時: フェードアウト → リスト再配置
    └─ 自動スクロール調整
```

#### 個別アイテム操作
```
[支出アイテムタップ]
    ↓
[詳細情報ポップアップ]
    ├─ 完全な支出情報表示
    ├─ レシート画像（添付時）
    ├─ メモ・タグ情報
    └─ [編集][削除][複製]ボタン

[スワイプアクション（モバイル）]
    ├─ 左スワイプ → [編集][削除]ボタン表示
    ├─ 右スワイプ → [複製][お気に入り]ボタン表示
    ├─ 長押し → コンテキストメニュー
    └─ ダブルタップ → クイック編集モード

[編集ボタンクリック]
    ↓
[編集モード遷移]
    ├─ 支出記録画面へ遷移
    ├─ 該当データをプリセット
    ├─ 編集モードフラグ設定
    └─ 戻り先情報保持

[削除ボタンクリック]
    ↓
[削除確認ダイアログ]
    ├─ 支出詳細の再表示
    ├─ "この操作は取り消せません"警告
    ├─ [削除][キャンセル]選択
    └─ 削除実行時のアニメーション
```

## アニメーション仕様

### 画面遷移
- **初期表示**: スケルトン → 実データ 各要素0.3s差分
- **データ更新**: 現在値 → 新値 カウンターアニメーション 1.5s
- **グラフ描画**: 順次描画 各要素0.5s間隔
- **ホバー効果**: 要素拡大・影拡張 0.2s ease

### インタラクション
- **タップフィードバック**: リップル効果 0.3s
- **スワイプアクション**: 要素スライド 0.25s ease-out
- **プルリフレッシュ**: 引っ張り連動回転 リアルタイム
- **エラー表示**: シェイク + フェードイン 0.5s
