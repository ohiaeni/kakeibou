# 支出登録画面詳細設計

## 画面概要

ユーザーが日々の支出を記録・編集するためのメイン機能画面です。カテゴリ選択、金額入力、日付設定、レシート管理などの機能を提供します。

## 画面要素詳細

### 入力フィールド

1. **カテゴリ選択**
   - 形式: ドロップダウン
   - デフォルト: "選択してください"
   - 選択肢: 食費、交通費、娯楽費、日用品、医療費、教育費、その他
   - 必須項目

2. **金額入力**
   - 形式: `<input type="number">`
   - 通貨記号: ¥ (自動付与)
   - バリデーション: 1円以上、1,000,000円以下
   - 必須項目
   - プレースホルダー: "0"

3. **支出日**
   - 形式: 日付ピッカー
   - デフォルト: 今日の日付
   - 範囲: 過去1年〜今日
   - 必須項目

4. **支払方法**
   - 形式: ドロップダウン
   - 選択肢: 現金、クレジットカード、デビットカード、電子マネー、QRコード決済
   - デフォルト: 現金
   - 任意項目

5. **店舗・場所**
   - 形式: `<input type="text">`
   - 最大文字数: 50文字
   - 任意項目
   - プレースホルダー: "店舗名または場所"

6. **メモ**
   - 形式: `<textarea>`
   - 最大文字数: 200文字
   - 任意項目
   - プレースホルダー: "詳細情報やメモ"

### コントロール要素

1. **レシート管理**
   - **撮影ボタン**: カメラ起動
   - **ファイル選択ボタン**: ファイルブラウザ起動
   - **添付ファイル表示**: サムネイル一覧

2. **操作ボタン**
   - **クリアボタン**: フォームリセット
   - **保存ボタン**: 支出記録保存

3. **今日の支出リスト**
   - **編集ボタン**: 該当記録をフォームに読み込み
   - **削除ボタン**: 確認ダイアログ後削除

## バリデーション仕様

### 必須項目チェック
- カテゴリ: 未選択時エラー
- 金額: 未入力または0円時エラー
- 支出日: 未選択時エラー

### 金額バリデーション
- 範囲: 1円〜1,000,000円
- 形式: 正の整数のみ
- エラーメッセージ: "1円以上1,000,000円以下で入力してください"

### 文字数制限
- 店舗・場所: 50文字以下
- メモ: 200文字以下
- エラーメッセージ: "○○文字以下で入力してください"

### ファイルバリデーション
- 対応形式: JPEG, PNG, PDF
- ファイルサイズ: 5MB以下
- 添付可能数: 5件まで

## 機能仕様

### レシート管理機能

#### 撮影機能
- カメラ起動
- プレビュー表示
- リサイズ・圧縮処理
- 自動回転補正

#### ファイル選択機能
- ファイルブラウザ起動
- ドラッグ&ドロップ対応
- 複数選択対応

#### OCR機能（将来拡張）
- レシート文字認識
- 金額自動入力
- 店舗名自動入力

### データ保存機能

#### 一時保存
- フォーム内容の自動保存（30秒間隔）
- ブラウザ閉じた場合の復元

#### 最終保存
- バリデーション実行
- API送信
- 成功時のフィードバック

### 履歴表示機能

#### 今日の支出
- 本日登録分のリアルタイム表示
- 時刻順ソート
- 合計金額表示

#### クイック操作
- 編集: フォームに値セット
- 削除: 確認ダイアログ表示
- 複製: 同一内容で新規作成
