# カテゴリー管理インタラクション

## インタラクション概要

支出登録画面でのカテゴリー選択、新規作成、編集、削除に関するユーザーインタラクションを定義します。

## カテゴリー選択インタラクション

### 基本選択操作
```
[カテゴリー選択UI]
    ├─ ドロップダウン形式
    ├─ グリッド形式（アイコン表示）
    ├─ リスト形式（詳細表示）
    └─ 検索ボックス付きセレクト

[選択操作フロー]
    ├─ カテゴリー一覧表示
    ├─ 選択項目のハイライト
    ├─ 選択確定処理
    ├─ 関連項目の自動更新
    └─ 選択状態の視覚的フィードバック

[よく使うカテゴリー機能]
    ├─ 使用頻度別ソート
    ├─ 上位5件の優先表示
    ├─ ワンクリック選択
    ├─ 学習機能による最適化
    └─ 時間帯別推奨表示
```

### カテゴリー階層表示
```
[階層構造の表現]
    ├─ 大カテゴリー表示
    ├─ 中カテゴリー展開
    ├─ 小カテゴリー詳細
    ├─ 階層レベルの視覚的区別
    └─ 展開/折りたたみ制御

[階層ナビゲーション]
    ├─ パンくずリスト表示
    ├─ 上位階層への戻り
    ├─ 階層ジャンプ機能
    ├─ 階層全体の可視化
    └─ 迷子防止機能

[スマートフィルタリング]
    ├─ キーワード検索
    ├─ タグベース検索
    ├─ 金額範囲絞り込み
    ├─ 使用頻度絞り込み
    └─ 日付範囲絞り込み
```

## 新規カテゴリー作成

### 作成トリガー
```
[作成開始操作]
    ├─ "+新しいカテゴリー"ボタン
    ├─ 検索結果なし時の作成提案
    ├─ 右クリックメニューから作成
    ├─ ショートカットキー（Ctrl+N）
    └─ 音声入力からの作成

[作成モード表示]
    ├─ インライン編集フォーム
    ├─ モーダルダイアログ
    ├─ サイドパネル展開
    ├─ 全画面エディター
    └─ ウィザード形式
```

### 作成フォーム設計
```
[必須入力項目]
    ├─ カテゴリー名（1-30文字）
    ├─ カテゴリー色（カラーピッカー）
    ├─ アイコン選択（プリセット一覧）
    ├─ 親カテゴリー選択
    └─ 表示順序設定

[任意設定項目]
    ├─ 説明文（最大100文字）
    ├─ 予算上限設定
    ├─ アラート設定
    ├─ タグ設定
    └─ 公開/非公開設定

[入力支援機能]
    ├─ カテゴリー名自動補完
    ├─ 類似カテゴリー検出
    ├─ 重複名称チェック
    ├─ 推奨設定値提案
    └─ テンプレートからの作成
```

### バリデーションと確認
```
[リアルタイムバリデーション]
    ├─ 文字数制限チェック
    ├─ 禁止文字チェック
    ├─ 重複名称チェック
    ├─ 階層深度制限チェック
    └─ 必須項目入力チェック

[作成前確認]
    ├─ 入力内容プレビュー
    ├─ 設定値の確認表示
    ├─ 類似カテゴリーとの比較
    ├─ 階層位置の確認
    └─ 影響範囲の説明

[作成完了処理]
    ├─ 成功メッセージ表示
    ├─ 新規カテゴリーの自動選択
    ├─ カテゴリー一覧の更新
    ├─ 最近使用一覧への追加
    └─ 統計情報の更新
```

## カテゴリー編集機能

### 編集開始操作
```
[編集トリガー]
    ├─ カテゴリー名ダブルクリック
    ├─ 右クリックメニュー"編集"
    ├─ 編集ボタンクリック
    ├─ ショートカットキー（F2）
    └─ 長押し操作（モバイル）

[編集モード表示]
    ├─ インライン編集
    ├─ ポップオーバー編集
    ├─ サイドパネル編集
    ├─ モーダル編集
    └─ 一括編集モード
```

### 編集項目とインタラクション
```
[基本情報編集]
    ├─ 名前変更
    │   ├─ リアルタイム重複チェック
    │   ├─ 履歴データとの整合性確認
    │   ├─ 変更影響範囲の表示
    │   └─ 変更確認ダイアログ
    ├─ 色変更
    │   ├─ カラーピッカー表示
    │   ├─ プリセット色選択
    │   ├─ カスタム色設定
    │   └─ プレビュー機能
    └─ アイコン変更
        ├─ アイコン一覧表示
        ├─ カスタムアイコンアップロード
        ├─ アイコンプレビュー
        └─ 推奨アイコン提案

[階層構造編集]
    ├─ 親カテゴリー変更
    ├─ 子カテゴリーの移動
    ├─ 階層レベルの調整
    ├─ ドラッグ&ドロップ操作
    └─ 階層整合性チェック
```

### 高度な編集機能
```
[一括編集機能]
    ├─ 複数選択
    ├─ 一括色変更
    ├─ 一括移動
    ├─ 一括削除
    └─ 一括設定変更

[履歴データとの連携]
    ├─ 過去データへの影響確認
    ├─ データ移行オプション
    ├─ 統計情報の再計算
    ├─ レポートへの影響表示
    └─ バックアップ作成提案
```

## カテゴリー削除機能

### 削除前確認プロセス
```
[削除可能性チェック]
    ├─ 使用履歴の確認
    ├─ 子カテゴリーの存在確認
    ├─ 関連データの影響範囲確認
    ├─ 削除制限項目チェック
    └─ システムカテゴリー保護

[削除確認ダイアログ]
    ├─ 削除対象の明示
    ├─ 影響範囲の詳細表示
    ├─ "○件の支出記録に影響があります"
    ├─ 代替カテゴリーの提案
    └─ 削除実行の最終確認

[データ処理オプション]
    ├─ 他カテゴリーへの移行
    ├─ 未分類カテゴリーへの移動
    ├─ データごと完全削除
    ├─ アーカイブ化（非表示）
    └─ 削除予約（日時指定）
```

### 削除実行と後処理
```
[削除実行プロセス]
    ├─ バックアップ作成
    ├─ 関連データの処理
    ├─ インデックス更新
    ├─ 統計情報再計算
    └─ 削除ログ記録

[削除後の状態管理]
    ├─ カテゴリー一覧から除外
    ├─ 削除済みマーク付与
    ├─ 履歴からの参照処理
    ├─ レポート表示調整
    └─ 復旧可能期間設定

[復旧機能]
    ├─ 削除取り消し（24時間以内）
    ├─ ゴミ箱機能
    ├─ 削除履歴表示
    ├─ 一括復旧機能
    └─ データ整合性チェック
```
