# 支出記録インタラクション

## インタラクション概要

ユーザーが支出情報を入力・編集・保存する際の画面動作を定義します。

## インタラクションフロー

### 1. 初期表示
```
[画面読み込み]
    ↓
[フォーム初期化]
    ├─ カテゴリ: "選択してください"
    ├─ 金額: 空白
    ├─ 支出日: 今日の日付
    ├─ 支払方法: "現金"
    ├─ 店舗・場所: 空白
    └─ メモ: 空白
    ↓
[今日の支出リスト読み込み]
    ↓
[一時保存データ復元チェック]
    ├─ 復元データあり → 復元確認ダイアログ表示
    └─ データなし → 通常フォーム表示
```

### 2. カテゴリ選択インタラクション
```
[カテゴリドロップダウンクリック]
    ↓
[選択肢展開]
    ├─ 食費 🍽️
    ├─ 交通費 🚃
    ├─ 娯楽費 🎮
    ├─ 日用品 🧴
    ├─ 医療費 💊
    ├─ 教育費 📚
    └─ その他 📦
    ↓
[選択]
    ├─ アイコン + テキスト表示
    ├─ ドロップダウン閉じる
    └─ 保存ボタン有効化チェック
```

### 3. 金額入力インタラクション
```
[金額フィールドフォーカス]
    ↓
[数値キーボード表示]（モバイル）
    ↓
[入力値監視]
    ├─ リアルタイム数値フォーマット
    ├─ カンマ区切り表示（例：1,200）
    └─ 通貨記号自動付与（¥1,200）

[バリデーション（リアルタイム）]
    ├─ 1未満 → "1円以上で入力してください"
    ├─ 1,000,000超過 → "1,000,000円以下で入力してください"
    ├─ 非数値 → "数値を入力してください"
    └─ 正常 → エラークリア

[フォーカスアウト]
    ↓
[フォーマット確定] → "¥1,200"形式で表示
```

### 4. 日付選択インタラクション
```
[日付フィールドクリック]
    ↓
[カレンダーピッカー表示]
    ├─ 今日の日付ハイライト
    ├─ 選択可能範囲表示（過去1年〜今日）
    └─ 無効日付グレーアウト

[日付選択]
    ├─ 未来日選択 → "未来の日付は選択できません"
    ├─ 過去1年超選択 → "過去1年以内の日付を選択してください"
    └─ 有効日付 → フィールドに反映・カレンダー閉じる

[今日ボタン]（カレンダー内）
    ↓
[本日日付設定] → カレンダー閉じる
```

### 5. レシート管理インタラクション
```
[撮影ボタンクリック]
    ↓
[カメラ権限チェック]
    ├─ 権限なし → 権限要求ダイアログ
    └─ 権限あり → カメラ起動

[カメラ画面]
    ├─ [撮影ボタン] → 写真撮影
    ├─ [キャンセル] → カメラ閉じる
    └─ [フラッシュ切替] → フラッシュON/OFF

[撮影完了]
    ↓
[プレビュー表示]
    ├─ [再撮影] → カメラ画面に戻る
    ├─ [キャンセル] → カメラ閉じる
    └─ [使用] → 画像処理・添付

[ファイル選択ボタンクリック]
    ↓
[ファイルブラウザ起動]
    ├─ 対応形式フィルタ（JPEG, PNG, PDF）
    └─ 複数選択可能

[ファイル添付処理]
    ├─ サイズチェック（5MB以下）
    ├─ 形式チェック
    ├─ サムネイル生成
    └─ 添付リストに追加

[添付ファイル表示]
    ├─ サムネイル表示
    ├─ ファイル名表示
    ├─ [×]削除ボタン
    └─ [👁️]プレビューボタン
```

### 6. フォーム操作インタラクション

#### 自動保存
```
[入力変更検知]
    ↓
[30秒タイマー開始]
    ↓
[タイマー満了]
    ├─ LocalStorageに保存
    ├─ 保存インジケーター表示
    └─ 3秒後インジケーター非表示
```

#### クリアボタン
```
[クリアボタンクリック]
    ↓
[確認ダイアログ表示]
    "入力内容をクリアしますか？"
    ├─ [キャンセル] → ダイアログ閉じる
    └─ [クリア] → フォームリセット

[フォームリセット]
    ├─ 全フィールドクリア
    ├─ 添付ファイル削除
    ├─ エラーメッセージクリア
    └─ 一時保存データ削除
```

#### 保存ボタン
```
[保存ボタンクリック]
    ↓
[最終バリデーション]
    ├─ エラーあり → エラーフィールドにフォーカス
    └─ 正常 → 保存処理開始

[保存処理]
    ├─ ボタン無効化
    ├─ "保存中..." テキスト変更
    ├─ スピナー表示
    └─ API送信

[API成功レスポンス]
    ├─ 成功メッセージ表示
    ├─ フォームクリア
    ├─ 今日の支出リスト更新
    └─ 一時保存データ削除

[API失敗レスポンス]
    ├─ エラーメッセージ表示
    ├─ ボタン再有効化
    └─ スピナー非表示
```

### 7. 今日の支出リストインタラクション

#### 編集操作
```
[編集ボタンクリック]
    ↓
[確認ダイアログ表示]
    "現在の入力内容は破棄されます。編集しますか？"
    ├─ [キャンセル] → ダイアログ閉じる
    └─ [編集] → データ読み込み

[データ読み込み]
    ├─ フォームに値セット
    ├─ 編集モード表示
    ├─ 保存ボタン → "更新"に変更
    └─ ID保持（更新用）
```

#### 削除操作
```
[削除ボタンクリック]
    ↓
[確認ダイアログ表示]
    "この支出記録を削除しますか？この操作は取り消せません。"
    ├─ [キャンセル] → ダイアログ閉じる
    └─ [削除] → 削除処理

[削除処理]
    ├─ API削除リクエスト
    ├─ 成功 → リストから除去・成功メッセージ
    └─ 失敗 → エラーメッセージ表示
```

#### 複製操作
```
[複製ボタンクリック]
    ↓
[データ複製]
    ├─ フォームに値セット（ID除く）
    ├─ 支出日 → 今日に変更
    ├─ 新規作成モード
    └─ 複製元情報表示
```

## 詳細動作仕様

### 保存ボタン有効化条件
```
[必須項目チェック]
├─ カテゴリ: 選択済み
├─ 金額: 1円以上入力済み
└─ 支出日: 選択済み

[全条件満足] → ボタン有効化（青色）
[条件不足] → ボタン無効状態（グレー）
```

### リアルタイムフィードバック

#### 金額入力時の視覚効果
```
[金額増加時]
    ├─ フィールド枠線: 緑色点滅
    ├─ 金額文字: 拡大アニメーション
    └─ 効果音: 小さなビープ音（設定により）

[高額入力時（10,000円以上）]
    ├─ 確認ダイアログ表示
    └─ "高額な支出ですが、間違いありませんか？"
```

#### 今日の支出合計表示
```
[支出登録/更新/削除時]
    ↓
[合計金額再計算]
    ├─ カウンターアニメーション
    ├─ 予算残高更新
    └─ 予算超過時警告表示
```

### エラーハンドリング

#### ネットワークエラー
```
[API通信エラー]
    ├─ オフライン検知 → "オフラインです。接続を確認してください"
    ├─ タイムアウト → "通信がタイムアウトしました"
    └─ サーバーエラー → "サーバーエラーが発生しました"

[オフライン時の動作]
    ├─ 入力データローカル保存
    ├─ オフライン表示
    └─ オンライン復帰時自動同期
```

#### ファイルアップロードエラー
```
[ファイルエラー分類]
├─ サイズ超過 → "ファイルサイズが5MBを超えています"
├─ 形式不正 → "対応していないファイル形式です"
├─ アップロード失敗 → "ファイルのアップロードに失敗しました"
└─ 数量超過 → "添付できるファイルは5件までです"
```

## アニメーション仕様

### フォーム操作
- **フィールドフォーカス**: 枠線色変更 0.2s ease
- **バリデーション成功**: チェックアイコン フェードイン 0.3s
- **エラー表示**: エラーメッセージ スライドダウン 0.3s
- **保存成功**: 成功バナー スライドイン 0.5s

### リスト操作
- **新規追加**: リストアイテム フェードイン 0.3s
- **削除**: リストアイテム スライドアウト 0.3s
- **編集モード**: ハイライト点滅 0.5s × 3回

### レシート操作
- **ファイル添付**: サムネイル ズームイン 0.3s
- **削除**: サムネイル フェードアウト 0.2s
- **プレビュー**: モーダル フェードイン 0.3s
