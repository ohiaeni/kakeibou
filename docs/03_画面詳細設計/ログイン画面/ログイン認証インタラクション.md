# ログイン認証インタラクション

## インタラクション概要

ユーザーがメールアドレスとパスワードを入力してログイン認証を行う際の画面動作を定義します。

## インタラクションフロー

### 1. 初期表示
```
[画面読み込み]
    ↓
[フォーム初期化]
    ↓
[入力フィールドフォーカス設定] → メールアドレス欄にフォーカス
```

### 2. 入力操作
```
[メールアドレス入力]
    ↓
[リアルタイムバリデーション]
    ├─ 有効 → アイコン表示（緑）
    └─ 無効 → エラーメッセージ表示（赤）

[パスワード入力]
    ↓
[文字数チェック]
    ├─ 8文字以上 → 次のフィールドへ移動可能
    └─ 8文字未満 → エラーメッセージ表示

[両フィールド有効]
    ↓
[ログインボタン有効化] → 色変更・クリック可能
```

### 3. ログイン実行
```
[ログインボタンクリック]
    ↓
[ローディング開始]
    ├─ ボタンテキスト変更: "ログイン中..."
    ├─ ボタン無効化
    └─ スピナー表示

[API認証リクエスト]
    ├─ 成功
    │   ├─ [トークン保存]
    │   ├─ [成功メッセージ表示] → "ログインしました"
    │   └─ [ダッシュボードへリダイレクト]
    │
    └─ 失敗
        ├─ [エラーメッセージ表示]
        ├─ [ローディング終了]
        └─ [フォームリセット]
```

## 詳細動作仕様

### 入力フィールドインタラクション

#### メールアドレス欄
- **フォーカス時**: 枠線色変更（#1976D2）
- **入力時**: リアルタイムでメール形式チェック
- **有効時**: 右側に緑色のチェックアイコン表示
- **無効時**: 右側に赤色のエラーアイコン表示
- **フォーカスアウト時**: エラーメッセージ表示（無効な場合）

#### パスワード欄
- **フォーカス時**: 枠線色変更（#1976D2）
- **入力時**: 文字数カウンター表示
- **表示切替アイコン**: 
  - クリック時: パスワード表示/非表示切り替え
  - アイコン変更: 目のアイコン ⇔ 目に斜線のアイコン

### ボタンインタラクション

#### ログインボタン
- **無効状態**: 
  - 背景色: #E0E0E0
  - テキスト色: #9E9E9E
  - クリック無効
- **有効状態**:
  - 背景色: #1976D2
  - テキスト色: #FFFFFF
  - ホバー時: 背景色 #1565C0
- **ローディング状態**:
  - テキスト: "ログイン中..."
  - スピナーアニメーション表示
  - クリック無効

### チェックボックスインタラクション

#### ログイン状態保持
- **チェック時**: ローカルストレージにフラグ保存
- **未チェック時**: セッションのみでトークン管理
- **視覚フィードバック**: チェックアニメーション

### リンクインタラクション

#### パスワードを忘れたリンク
- **ホバー時**: テキスト装飾（下線）
- **クリック時**: パスワードリセット画面へ遷移

#### 新規登録リンク
- **ホバー時**: テキスト装飾（下線）
- **クリック時**: ユーザー登録画面へ遷移

## エラーハンドリング

### バリデーションエラー
```
[エラー検出]
    ↓
[フィールド枠線色変更] → 赤色（#F44336）
    ↓
[エラーメッセージ表示] → フィールド下部に赤色テキスト
    ↓
[エラー解消まで状態維持]
```

### 認証エラー
```
[認証失敗レスポンス]
    ↓
[エラー分類]
    ├─ 認証情報不正 → "メールアドレスまたはパスワードが正しくありません"
    ├─ アカウント無効 → "アカウントが無効です。管理者にお問い合わせください"
    ├─ ログイン制限 → "ログイン試行回数が上限に達しました。しばらくしてからお試しください"
    └─ サーバーエラー → "サーバーエラーが発生しました。しばらくしてからお試しください"
    ↓
[エラーメッセージ表示] → 画面上部にアラート表示
    ↓
[5秒後自動非表示] または [×ボタンクリックで非表示]
```

## アニメーション仕様

### フェードイン
- **対象**: エラーメッセージ、成功メッセージ
- **時間**: 0.3秒
- **イージング**: ease-in-out

### スライドアニメーション
- **対象**: 画面遷移
- **時間**: 0.5秒
- **方向**: 左からスライドイン

### ローディングスピナー
- **回転速度**: 1秒/回転
- **色**: #1976D2
- **サイズ**: 16px

## レスポンシブ対応

### モバイル表示時の変更点
- **フォントサイズ**: 16px以上（ズーム防止）
- **ボタンサイズ**: 最小48px高さ
- **タッチターゲット**: 最小44px×44px
- **キーボード表示時**: 画面レイアウト調整
