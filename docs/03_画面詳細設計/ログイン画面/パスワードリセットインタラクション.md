# パスワードリセットインタラクション

## インタラクション概要

ユーザーがパスワードを忘れた場合のリセット手続きに関する画面動作を定義します。

## インタラクションフロー

### 1. パスワードリセット開始
```
[パスワードを忘れたリンククリック]
    ↓
[パスワードリセット画面遷移]
    ├─ スライドトランジション
    ├─ フォーム初期化
    └─ メールアドレス欄にフォーカス
```

### 2. メールアドレス入力・送信
```
[メールアドレス入力]
    ↓
[リアルタイムバリデーション]
    ├─ メール形式チェック
    ├─ 登録済みアドレスチェック
    └─ 送信ボタン有効化判定

[送信ボタンクリック]
    ↓
[送信処理開始]
    ├─ ボタン無効化
    ├─ "送信中..." 表示
    ├─ スピナー表示
    └─ API送信

[送信成功]
    ├─ 成功メッセージ表示
    ├─ "確認メールを送信しました"
    ├─ メール受信確認画面表示
    └─ 自動カウントダウン開始

[送信失敗]
    ├─ エラーメッセージ表示
    ├─ "メールアドレスが見つかりません"
    └─ 再入力促進
```

### 3. メール受信確認
```
[確認画面表示]
    ├─ 送信先メールアドレス表示
    ├─ 受信確認手順説明
    ├─ 迷惑メールフォルダ確認案内
    └─ 再送信ボタン表示

[再送信ボタン]
    ├─ クールダウンタイマー（60秒）
    ├─ カウントダウン表示
    ├─ タイマー終了後有効化
    └─ 再送信処理実行

[メールリンククリック]
    ↓
[パスワードリセットページ遷移]
    ├─ トークン検証
    ├─ 有効期限チェック
    └─ 新パスワード入力画面表示
```

### 4. 新パスワード設定
```
[新パスワード入力画面]
    ├─ 新パスワード入力欄
    ├─ パスワード確認欄
    ├─ パスワード強度インジケーター
    └─ 設定ボタン

[パスワード強度チェック]
    ├─ 8文字以上チェック
    ├─ 英数字含有チェック
    ├─ 特殊文字推奨表示
    └─ 強度バー更新（弱・中・強）

[パスワード確認入力]
    ├─ 一致チェック（リアルタイム）
    ├─ 一致時：緑色チェックマーク
    └─ 不一致時：エラーメッセージ

[パスワード設定実行]
    ├─ 最終バリデーション
    ├─ API送信
    ├─ 成功：完了画面表示
    └─ 失敗：エラー処理
```

### 5. リセット完了
```
[パスワードリセット完了]
    ├─ 成功メッセージ表示
    ├─ "パスワードが正常に更新されました"
    ├─ ログイン画面への自動遷移（5秒後）
    └─ [今すぐログイン]ボタン

[セキュリティ通知]
    ├─ パスワード変更通知メール送信
    ├─ 全セッション無効化
    ├─ ログイン履歴記録
    └─ セキュリティログ更新
```

## エラーハンドリング

### トークン関連エラー
```
[無効なトークン]
    ├─ "リンクが無効です"
    ├─ 再申請ボタン表示
    └─ パスワードリセット画面へ誘導

[期限切れトークン]
    ├─ "リンクの有効期限が切れています"
    ├─ "24時間以内にリセットしてください"
    └─ 新規申請案内

[使用済みトークン]
    ├─ "このリンクは既に使用されています"
    ├─ 既にリセット済み案内
    └─ ログイン画面への誘導
```

### ネットワークエラー
```
[通信エラー]
    ├─ "通信エラーが発生しました"
    ├─ リトライボタン表示
    ├─ オフライン状態検知
    └─ 復旧時自動リトライ

[サーバーエラー]
    ├─ "一時的な問題が発生しています"
    ├─ "しばらくしてからお試しください"
    └─ サポート連絡先表示
```

## セキュリティ機能

### 不正アクセス対策
```
[連続試行制限]
    ├─ 同一IPから5回/時間制限
    ├─ 制限時一時的ブロック
    ├─ ブロック解除カウントダウン
    └─ 管理者通知機能

[トークンセキュリティ]
    ├─ ランダム生成（256bit）
    ├─ 24時間有効期限
    ├─ 1回使用で無効化
    └─ ブルートフォース攻撃対策
```

## アニメーション仕様

### 画面遷移
- **フォーム切替**: スライド遷移 0.5s ease-in-out
- **成功表示**: フェードイン + チェックマーク 0.8s
- **エラー表示**: シェイク + 赤色点滅 0.5s
- **カウントダウン**: 数字変更アニメーション 1.0s
