# 設定画面詳細設計

## 画面概要

ユーザーがアプリケーションの各種設定を管理するための画面です。アカウント設定、カテゴリ管理、通知設定、データ管理機能を提供します。

## バリデーション仕様

### アカウント設定
- **ユーザー名**: 2-20文字、日本語・英数字
- **メールアドレス**: RFC準拠、重複チェック
- **パスワード**: 8文字以上、英数字含む

### カテゴリ管理
- **カテゴリ名**: 1-20文字、重複禁止
- **最大カテゴリ数**: 20個まで
- **削除制限**: 使用済みカテゴリの削除警告

### 通知設定
- **通知時間**: 24時間形式
- **メール設定**: 有効なメールアドレス必須

## セキュリティ仕様

### パスワード変更
1. **現在パスワード確認**
2. **新パスワード入力**（2回）
3. **強度チェック**
4. **変更通知メール送信**

### セッション管理
1. **アクティブセッション表示**
2. **デバイス・ブラウザ情報**
3. **最終アクセス時刻**
4. **個別/一括セッション終了**

### 二段階認証
1. **TOTP対応**（Google Authenticator等）
2. **バックアップコード生成**
3. **SMS認証**（オプション）
4. **復旧手順**
