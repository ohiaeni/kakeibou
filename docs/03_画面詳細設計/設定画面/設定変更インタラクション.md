# 設定変更インタラクション

## インタラクション概要

ユーザーが各種設定を変更・保存・管理する際の画面動作を定義します。

## インタラクションフロー

### 1. 初期表示・ナビゲーション
```
[設定画面読み込み]
    ↓
[サイドバーナビゲーション表示]
    ├─ デフォルト選択: アカウント設定
    ├─ 現在設定値の取得・表示
    └─ 権限チェック（表示制御）
    ↓
[設定データ読み込み]
    ├─ ユーザー設定取得
    ├─ システム設定取得
    └─ フォーム初期化

[サイドバーナビゲーション]
    ├─ [設定項目クリック] → コンテンツエリア切替
    ├─ アクティブ項目ハイライト
    ├─ 未保存変更警告（切替時）
    └─ ブレッドクラム更新
```

### 2. アカウント設定インタラクション

#### ユーザー名変更
```
[ユーザー名フィールドフォーカス]
    ↓
[現在値表示・編集可能化]
    ↓
[入力値監視]
    ├─ リアルタイムバリデーション
    ├─ 文字数カウンター表示
    └─ 重複チェック（デバウンス: 500ms）

[バリデーションロジック]
    ├─ 2文字未満 → "2文字以上で入力してください"
    ├─ 20文字超過 → "20文字以下で入力してください"
    ├─ 不正文字 → "使用できない文字が含まれています"
    ├─ 重複 → "このユーザー名は既に使用されています"
    └─ 正常 → 緑色チェックアイコン表示

[フォーカスアウト]
    ├─ 最終バリデーション
    ├─ 変更フラグ設定
    └─ 保存ボタン有効化
```

#### パスワード変更
```
[パスワード変更ボタンクリック]
    ↓
[パスワード変更モーダル表示]
    ├─ 現在パスワード入力欄
    ├─ 新パスワード入力欄
    ├─ 新パスワード確認欄
    └─ パスワード強度インジケーター

[現在パスワード入力]
    ↓
[認証チェック（リアルタイム）]
    ├─ 正しい → 緑色チェック表示
    └─ 間違い → "現在のパスワードが正しくありません"

[新パスワード入力]
    ↓
[強度評価]
    ├─ 文字数チェック（8文字以上）
    ├─ 文字種チェック（英数字）
    ├─ 複雑度評価
    └─ 強度バー更新（弱/中/強）

[パスワード確認入力]
    ├─ 一致チェック（リアルタイム）
    ├─ 一致 → "✓ パスワードが一致しています"
    └─ 不一致 → "パスワードが一致しません"

[変更実行]
    ├─ 最終バリデーション
    ├─ API送信
    ├─ 成功 → 確認メール送信・モーダル閉じる
    └─ 失敗 → エラーメッセージ表示
```

#### アバター画像変更
```
[アバター画像エリアクリック] または [変更ボタンクリック]
    ↓
[画像選択ダイアログ表示]
    ├─ [ファイルから選択]
    ├─ [カメラで撮影]（モバイル）
    ├─ [アバター作成ツール]
    └─ [デフォルトアバター選択]

[ファイル選択]
    ↓
[画像バリデーション]
    ├─ ファイル形式チェック（JPEG, PNG）
    ├─ ファイルサイズチェック（5MB以下）
    ├─ 画像サイズチェック（最小100x100px）
    └─ 正常 → プレビュー表示

[画像プレビュー・編集]
    ├─ クロップ機能
    ├─ 回転機能
    ├─ ズーム機能
    └─ フィルター適用

[アップロード処理]
    ├─ プログレスバー表示
    ├─ 画像圧縮・最適化
    ├─ API送信
    └─ 成功 → プロフィール画像更新・プレビュー閉じる
```

### 3. カテゴリ管理インタラクション

#### カテゴリ一覧表示
```
[カテゴリ管理選択]
    ↓
[カテゴリ一覧取得・表示]
    ├─ 使用統計付き表示
    ├─ ドラッグ&ドロップ並び替え対応
    └─ 削除可能性判定

[並び替え操作]
    ├─ [ドラッグ開始] → 半透明表示
    ├─ [ドロップ位置表示] → 挿入ライン表示
    ├─ [ドロップ] → 位置更新・アニメーション
    └─ [並び順保存] → API送信
```

#### 新規カテゴリ追加
```
[新規追加ボタンクリック]
    ↓
[追加フォーム表示]
    ├─ カテゴリ名入力欄（フォーカス）
    ├─ アイコン選択（デフォルト設定）
    ├─ カラー選択（デフォルト設定）
    └─ プレビュー表示

[カテゴリ名入力]
    ↓
[リアルタイムバリデーション]
    ├─ 文字数チェック（1-20文字）
    ├─ 重複チェック
    ├─ 禁止文字チェック
    └─ プレビュー更新

[アイコン選択]
    ↓
[アイコンピッカー表示]
    ├─ カテゴリ別グループ表示
    ├─ 検索機能
    ├─ 最近使用したアイコン
    └─ 選択 → プレビュー更新

[カラー選択]
    ├─ プリセットカラー
    ├─ カスタムカラーピッカー
    ├─ カラーコード入力
    └─ アクセシビリティチェック

[追加実行]
    ├─ バリデーション
    ├─ API送信
    ├─ 成功 → リスト追加・フォームリセット
    └─ 失敗 → エラー表示
```

#### カテゴリ編集・削除
```
[編集ボタンクリック]
    ↓
[インライン編集モード]
    ├─ 編集フィールド表示
    ├─ 元データ保持
    └─ [保存][キャンセル]ボタン表示

[削除ボタンクリック]
    ↓
[削除可能性チェック]
    ├─ 使用中データ確認
    ├─ 使用中 → 警告ダイアログ表示
    └─ 未使用 → 確認ダイアログ表示

[削除確認ダイアログ]
    ├─ 使用統計表示
    ├─ 影響範囲説明
    ├─ 代替カテゴリ選択（使用中の場合）
    └─ [削除][キャンセル]
```

### 4. 通知設定インタラクション

#### 通知権限管理
```
[ブラウザ通知設定変更]
    ↓
[権限状態チェック]
    ├─ 未許可 → 権限要求
    ├─ 拒否済み → 手動設定案内
    └─ 許可済み → 設定変更

[権限要求処理]
    ├─ Notification.requestPermission()
    ├─ 許可 → 設定有効化
    ├─ 拒否 → 設定無効化・案内表示
    └─ 無視 → 再要求提案
```

#### 通知スケジュール設定
```
[通知時間設定]
    ↓
[時間ピッカー表示]
    ├─ 24時間形式
    ├─ 15分間隔選択
    ├─ プリセット時間提案
    └─ タイムゾーン考慮

[リマインダー設定]
    ├─ 頻度選択（毎日/週次/月次）
    ├─ 曜日選択（週次の場合）
    ├─ 日付選択（月次の場合）
    └─ スヌーズ設定

[通知テスト機能]
    ├─ [テスト通知送信]ボタン
    ├─ 設定内容でテスト実行
    ├─ 通知表示確認
    └─ 結果フィードバック
```

### 5. データ管理インタラクション

#### データエクスポート
```
[エクスポートボタンクリック]
    ↓
[エクスポート設定ダイアログ]
    ├─ 形式選択（CSV/Excel/JSON）
    ├─ 期間指定（全期間/カスタム）
    ├─ データ種別選択
    └─ 文字エンコーディング選択

[エクスポート実行]
    ├─ プログレスバー表示
    ├─ データ生成（バックグラウンド）
    ├─ ダウンロード準備
    └─ 自動ダウンロード開始

[エクスポート完了]
    ├─ 成功通知表示
    ├─ ファイル情報表示（サイズ・件数）
    ├─ 再ダウンロードリンク
    └─ エクスポート履歴記録
```

#### データインポート
```
[ファイル選択] または [ドラッグ&ドロップ]
    ↓
[ファイルバリデーション]
    ├─ 形式チェック（CSV/Excel）
    ├─ サイズチェック（10MB以下）
    ├─ 文字エンコーディング検出
    └─ 正常 → プレビュー表示

[データプレビュー]
    ├─ 列マッピング設定
    ├─ データ型検証
    ├─ 重複データ検出
    ├─ エラー行ハイライト
    └─ インポート件数表示

[インポート実行]
    ├─ バリデーション実行
    ├─ プログレスバー表示
    ├─ バッチ処理実行
    ├─ エラーハンドリング
    └─ 結果サマリー表示

[インポート結果]
    ├─ 成功件数・失敗件数表示
    ├─ エラーログダウンロード
    ├─ データ検証提案
    └─ 確認・取り消し選択肢
```

#### データ削除
```
[データ削除ボタンクリック]
    ↓
[削除範囲選択ダイアログ]
    ├─ 期間指定削除
    ├─ カテゴリ指定削除
    ├─ 条件指定削除
    └─ 全データ削除

[削除確認プロセス]
    ├─ 影響範囲表示（件数・データ量）
    ├─ バックアップ作成提案
    ├─ 復元不可警告表示
    └─ 確認文字列入力要求

[削除実行]
    ├─ 最終確認ダイアログ
    ├─ パスワード再認証
    ├─ 削除処理実行
    ├─ プログレス表示
    └─ 完了通知・画面リフレッシュ
```

### 6. セキュリティ設定インタラクション

#### 二段階認証設定
```
[二段階認証有効化ボタン]
    ↓
[設定ガイダンス表示]
    ├─ QRコード生成・表示
    ├─ 手動入力用キー表示
    ├─ アプリ推奨リスト
    └─ 設定手順説明

[認証アプリ設定完了後]
    ↓
[確認コード入力要求]
    ├─ 6桁数字入力欄
    ├─ リアルタイム形式チェック
    ├─ 有効期限表示
    └─ 再生成機能

[確認成功]
    ├─ バックアップコード生成・表示
    ├─ 印刷・保存推奨
    ├─ 設定完了通知
    └─ 状態表示更新

[二段階認証無効化]
    ├─ パスワード再認証
    ├─ 確認コード入力
    ├─ 無効化確認ダイアログ
    └─ 設定解除・通知送信
```

#### セッション管理
```
[セッション一覧表示]
    ├─ アクティブセッション表示
    ├─ デバイス情報表示
    ├─ 位置情報表示（IP地域）
    ├─ 最終アクセス時刻
    └─ 現在セッション明示

[個別セッション終了]
    ├─ 確認ダイアログ表示
    ├─ セッション無効化
    ├─ 対象デバイスからログアウト
    └─ 一覧更新

[全セッション終了]
    ├─ 重要な操作の警告表示
    ├─ パスワード再認証
    ├─ 全セッション無効化
    ├─ 自動ログアウト（他デバイス）
    └─ 新規ログイン要求
```

## エラーハンドリング

### バリデーションエラー
```
[入力値エラー]
    ├─ フィールドレベル → 該当欄下部に表示
    ├─ フォームレベル → 画面上部バナー表示
    ├─ 自動非表示 → 修正時・5秒後
    └─ アクセシビリティ → スクリーンリーダー対応

[システムエラー]
    ├─ ネットワークエラー → リトライ機能
    ├─ 権限エラー → ログイン画面遷移
    ├─ サーバーエラー → エラーページ表示
    └─ 不明エラー → お問い合わせ案内
```

### データ整合性エラー
```
[設定競合検出]
    ├─ 他セッションでの変更検出
    ├─ 競合内容表示
    ├─ 手動マージ選択肢提供
    └─ 自動解決・手動解決選択

[権限変更検出]
    ├─ 設定権限の変更検出
    ├─ 制限事項説明
    ├─ 可能な操作の案内
    └─ 管理者連絡手段提供
```

## アニメーション仕様

### 設定変更フィードバック
- **保存成功**: 緑色チェックマーク フェードイン 0.5s
- **エラー**: 赤色アラート + 軽微シェイク 0.3s
- **変更検出**: フィールド枠線色変更 0.2s
- **処理中**: プログレスバー + スピナー

### ナビゲーション
- **タブ切替**: スライド遷移 0.3s ease-in-out
- **モーダル表示**: フェードイン + スケール 0.4s
- **アコーディオン**: 高さ変更 0.3s ease
- **ドロップダウン**: スライドダウン 0.2s
