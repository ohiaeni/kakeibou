# 支出履歴画面詳細設計

## 画面概要

ユーザーが過去の支出記録を検索・表示・管理するための画面です。期間絞り込み、カテゴリフィルター、並び替え機能を提供します。

## 画面要素詳細

### 検索・フィルター機能

1. **検索ワード入力**
   - 形式: `<input type="text">`
   - 対象: 店舗名、メモ内容
   - リアルタイム検索（デバウンス: 300ms）
   - プレースホルダー: "店舗名やメモで検索"

2. **期間選択**
   - 形式: 日付範囲ピッカー
   - プリセット: 今日、昨日、今週、今月、先月、過去3ヶ月
   - カスタム期間: 開始日〜終了日指定
   - デフォルト: 今月

3. **カテゴリフィルター**
   - 形式: マルチセレクトドロップダウン
   - 選択肢: 全て、食費、交通費、娯楽費、日用品、医療費、教育費、その他
   - デフォルト: "全て"

4. **並び順設定**
   - 選択肢: 
     - 日付順（新しい順・古い順）
     - 金額順（高い順・安い順）
     - カテゴリ順（ABC順）
   - デフォルト: 日付順（新しい順）

### 一覧表示機能

1. **データ表示項目**
   - 日付・曜日・時刻
   - カテゴリアイコン + 名称
   - 金額（通貨フォーマット）
   - 店舗・場所名
   - メモ（省略表示）
   - 添付ファイル有無アイコン

2. **サマリー表示**
   - 表示期間
   - 総支出金額
   - 記録件数
   - カテゴリ別集計（折りたたみ可能）

3. **ページネーション**
   - 1ページ: 20件表示
   - ページ番号表示
   - 前へ・次へボタン
   - 全件数表示

### 操作機能

1. **個別操作**
   - **編集**: 支出登録画面で編集
   - **削除**: 確認ダイアログ後削除
   - **複製**: 同一内容で新規作成
   - **詳細表示**: モーダルで詳細表示

2. **一括操作**
   - **チェックボックス選択**: 複数選択
   - **一括削除**: 選択項目の削除
   - **CSV出力**: 検索条件に応じたデータ出力

3. **エクスポート機能**
   - **CSV形式**: Excel互換
   - **期間指定**: 検索条件反映
   - **項目選択**: 出力項目カスタマイズ

## バリデーション仕様

### 期間検索
- 開始日 ≤ 終了日のチェック
- 最大検索期間: 1年間
- 未来日付の制限

### 検索条件
- 検索ワード: 50文字以下
- 期間: 必須選択
- エラーメッセージ: "正しい検索条件を入力してください"

## パフォーマンス仕様

### データ読み込み
- **初期表示**: 最新20件を高速表示
- **遅延読み込み**: スクロール時の追加読み込み
- **キャッシュ**: 表示済みデータの保持
- **プリロード**: 次ページの先読み

### 検索最適化
- **インデックス活用**: 日付・カテゴリでの高速検索
- **結果キャッシュ**: 同一条件での再利用
- **部分検索**: 大量データでの段階的表示
