# 予算アラートインタラクション

## インタラクション概要

予算設定画面での各種アラート設定、通知管理、警告表示に関するユーザーインタラクションを定義します。

## アラート設定インタラクション

### 基本アラート設定
```
[アラート種別設定]
    ├─ 予算超過アラート
    │   ├─ 100%超過時（予算完全消化）
    │   ├─ 110%超過時（予算オーバー）
    │   ├─ 125%超過時（大幅超過）
    │   └─ カスタム閾値設定
    ├─ 予算到達アラート
    │   ├─ 50%到達時（中間地点）
    │   ├─ 75%到達時（注意レベル）
    │   ├─ 90%到達時（警告レベル）
    │   └─ 段階的通知設定
    ├─ 残額アラート
    │   ├─ 残り10,000円以下
    │   ├─ 残り5,000円以下
    │   ├─ 残り1,000円以下
    │   └─ 金額指定アラート
    └─ 期間アラート
        ├─ 月末5日前
        ├─ 月末3日前
        ├─ 月末1日前
        └─ 期間カスタム設定

[通知タイミング設定]
    ├─ リアルタイム通知
    │   ├─ 支出記録時即座
    │   ├─ 予算更新時即座
    │   ├─ 設定変更時即座
    │   └─ データ同期時
    ├─ 定期通知
    │   ├─ 毎日朝8時
    │   ├─ 毎週月曜日
    │   ├─ 月初・月末
    │   └─ カスタムスケジュール
    └─ 条件付き通知
        ├─ 平日のみ
        ├─ 営業時間内のみ
        ├─ 位置情報連動
        └─ 使用パターン学習
```
