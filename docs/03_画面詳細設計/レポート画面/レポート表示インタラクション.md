# レポート表示インタラクション

## インタラクション概要

ユーザーがレポート画面でデータ分析結果を表示・操作・カスタマイズする際の画面動作を定義します。

## インタラクションフロー

### 1. 初期表示
```
[画面読み込み]
    ↓
[デフォルト設定適用]
    ├─ 分析期間: 今月
    ├─ 表示形式: 月次
    └─ カテゴリ: 全て
    ↓
[データ取得・分析処理]
    ├─ スケルトン画面表示
    ├─ プログレスバー表示
    └─ 段階的コンテンツ表示
    ↓
[グラフ描画・分析表示]
    ├─ 4つのチャート順次表示
    ├─ 分析サマリー生成
    └─ 改善提案表示
```

### 2. 期間選択インタラクション
```
[期間選択ドロップダウンクリック]
    ↓
[選択肢展開]
    ├─ 今日
    ├─ 今週
    ├─ 今月 ← (デフォルト)
    ├─ 先月
    ├─ 過去3ヶ月
    ├─ 過去6ヶ月
    ├─ 過去1年
    └─ カスタム期間...

[プリセット期間選択]
    ├─ ローディング開始
    ├─ 全グラフデータ再取得
    ├─ 分析処理再実行
    └─ 表示更新（フェード効果）

[カスタム期間選択]
    ↓
[カレンダーモーダル表示]
    ├─ 開始日・終了日選択
    ├─ 期間バリデーション
    ├─ プリセット保存オプション
    └─ [適用][キャンセル]

[期間変更時の処理]
    ├─ グラフアニメーション停止
    ├─ データフェッチ開始
    ├─ UI無効化（ローディング中）
    ├─ エラーハンドリング
    └─ 完了時UI再有効化
```

### 3. 表示形式変更インタラクション
```
[表示形式選択]
    ├─ 日次 → 日別データ集計
    ├─ 週次 → 週別データ集計  
    ├─ 月次 → 月別データ集計
    └─ 年次 → 年別データ集計

[集計レベル変更処理]
    ↓
[データ再集計]
    ├─ サーバーサイド集計処理
    ├─ クライアント側キャッシュ確認
    ├─ 必要データのみ再取得
    └─ グラフ軸・ラベル調整

[グラフタイプ自動調整]
    ├─ 日次 → 折れ線グラフ
    ├─ 週次 → 棒グラフ
    ├─ 月次 → 棒グラフ
    └─ 年次 → 面グラフ

[アニメーション制御]
    ├─ 既存グラフフェードアウト（0.3s）
    ├─ 軸・ラベル更新
    ├─ 新データプロット
    └─ 新グラフフェードイン（0.5s）
```

### 4. グラフインタラクションシステム

#### ホバー・タッチインタラクション
```
[グラフ要素ホバー]
    ↓
[ツールチップ表示]
    ├─ データポイント詳細
    ├─ 日付・時刻情報
    ├─ 金額・パーセンテージ
    └─ 比較データ（前期比等）

[ツールチップ内容（チャート別）]
├─ 棒グラフ: 期間 | 金額 | 予算比
├─ 円グラフ: カテゴリ | 金額 | 割合 | 件数
├─ 線グラフ: 日付 | 金額 | 移動平均
└─ レーダー: カテゴリ | 達成率 | 予算額

[マルチタッチ対応]（モバイル）
    ├─ シングルタップ → ツールチップ表示
    ├─ ダブルタップ → 詳細モーダル
    ├─ ピンチ → ズーム操作
    └─ スワイプ → パン操作
```

#### ズーム・パンインタラクション
```
[ズーム操作]
    ├─ マウスホイール → Y軸方向ズーム
    ├─ Ctrl+ホイール → X軸方向ズーム
    ├─ ピンチジェスチャー → 比例ズーム
    └─ ダブルクリック → リセット

[パン操作]
    ├─ ドラッグ操作 → 表示範囲移動
    ├─ 境界制限 → データ範囲内のみ
    ├─ 慣性スクロール → 自然な操作感
    └─ スナップ機能 → 重要ポイントに吸着

[ズーム状態管理]
    ├─ ズームレベル保持
    ├─ 表示範囲保持
    ├─ リセット機能
    └─ ブックマーク機能
```

### 5. フィルタリングインタラクション

#### カテゴリフィルター
```
[カテゴリ選択チェックボックス]
    ↓
[選択状態変更]
    ├─ 個別選択 → 該当データハイライト
    ├─ 複数選択 → 選択データのみ表示
    ├─ 全選択解除 → 空状態表示
    └─ 全選択 → 全データ表示

[フィルター適用アニメーション]
    ├─ 非選択要素フェードアウト（0.3s）
    ├─ 選択要素強調表示
    ├─ グラフ再描画
    └─ 統計値再計算・表示

[レジェンドインタラクション]
    ├─ クリック → カテゴリ表示切替
    ├─ ダブルクリック → 単独表示
    ├─ Shift+クリック → 範囲選択
    └─ Ctrl+クリック → 追加選択
```

#### 動的フィルタリング
```
[リアルタイムフィルター]
    ├─ 金額範囲スライダー
    ├─ 日付範囲選択
    ├─ 店舗・場所絞込
    └─ キーワード検索

[フィルター連動制御]
    ├─ 複数フィルター AND条件
    ├─ フィルター履歴管理
    ├─ クリア機能
    └─ 保存・読込機能
```

## エラーハンドリング

### データ取得エラー
```
[API通信エラー]
    ├─ ネットワークエラー → リトライ機能
    ├─ タイムアウト → 部分データ表示
    ├─ サーバーエラー → エラー画面表示
    └─ 認証エラー → ログイン画面遷移

[データ不整合エラー]
    ├─ 期間データなし → 空状態表示
    ├─ 部分データ欠損 → 警告付き表示
    ├─ 分析失敗 → 基本グラフのみ表示
    └─ フォーマットエラー → デフォルト表示
```

### グラフ描画エラー
```
[描画エラーハンドリング]
    ├─ Canvas未対応 → SVG代替表示
    ├─ メモリ不足 → データ量削減
    ├─ レンダリング失敗 → テキスト表示
    └─ ブラウザ制限 → 機能縮退
```

## アニメーション仕様

### グラフアニメーション
- **データ更新**: 要素間補間 1.0s ease-out
- **表示切替**: フェードイン/アウト 0.5s
- **ホバー効果**: スケール変更 0.2s ease
- **ズーム**: トランジション 0.3s ease-in-out

### UI フィードバック
- **ローディング**: スピナー + プログレスバー
- **成功**: 緑色チェックマーク 0.5s
- **エラー**: 赤色アラート + シェイク 0.3s
- **更新**: パルス効果 2.0s
