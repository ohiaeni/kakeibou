# チャート操作インタラクション

## インタラクション概要

レポート画面でのチャート表示、操作、カスタマイズ、エクスポートに関するユーザーインタラクションを定義します。

## チャート基本操作

### チャート種別選択
```
[チャートタイプ選択]
    ├─ 円グラフ（パイチャート）
    │   ├─ カテゴリー別支出割合
    │   ├─ 月別収支割合
    │   ├─ 支払い方法別割合
    │   └─ カスタム項目別割合
    ├─ 棒グラフ（バーチャート）
    │   ├─ 月別支出推移
    │   ├─ カテゴリー別比較
    │   ├─ 週別支出トレンド
    │   └─ 年別比較
    ├─ 線グラフ（ラインチャート）
    │   ├─ 支出トレンド分析
    │   ├─ 予算達成推移
    │   ├─ 残高推移
    │   └─ 節約効果推移
    ├─ 面グラフ（エリアチャート）
    │   ├─ 累積支出推移
    │   ├─ カテゴリー構成変化
    │   ├─ 収支バランス推移
    │   └─ 予算消化推移
    └─ 複合チャート
        ├─ 棒グラフ+線グラフ組み合わせ
        ├─ 複数軸表示
        ├─ 異なるデータ系列
        └─ 相関関係表示

[チャート切り替え操作]
    ├─ ドロップダウン選択
    ├─ タブ切り替え
    ├─ アイコンボタン選択
    ├─ ショートカットキー
    └─ ジェスチャー操作
```
