## 画面概要

新しいカテゴリを作成するための画面です。カテゴリ名、色、アイコンなどを設定できます。

## 画面機能

- カテゴリ名の入力・バリデーション
- カテゴリ色の選択
- アイコンの選択
- 重複チェック
- 入力データの保存
- 既存カテゴリの一覧表示

## UI要素詳細

### フォーム要素

| 要素名     | タイプ         | 必須 | 制約                 | 説明                   |
| ---------- | -------------- | ---- | -------------------- | ---------------------- |
| カテゴリ名 | テキスト       | ✓    | 最大50文字、重複不可 | カテゴリの名前         |
| カテゴリ色 | 選択           | ✓    | 8色から選択          | カテゴリを識別する色   |
| アイコン   | 選択           | ✓    | 16種類から選択       | カテゴリを表すアイコン |
| 説明       | テキストエリア |      | 最大200文字          | カテゴリの詳細説明     |

### バリデーション

- **カテゴリ名**
  - 必須入力
  - 最大 50 文字
  - 同名カテゴリの重複不可
  - 特殊文字の制限（英数字、ひらがな、カタカナ、漢字のみ）

- **カテゴリ色**
  - 必須選択
  - 8 色のうち 1 色を選択

- **アイコン**
  - 必須選択
  - 16 種類のうち 1 つを選択

- **説明**
  - 任意入力
  - 最大 200 文字

## 画面遷移

- **登録ボタン押下**
  - バリデーション成功 → カテゴリ一覧画面へ遷移（成功メッセージ表示）
  - バリデーション失敗 → 同画面でエラーメッセージ表示

- **キャンセルボタン押下**
  - カテゴリ一覧画面へ遷移（確認ダイアログなし）

- **戻るボタン押下（モバイル）**
  - カテゴリ一覧画面へ遷移
