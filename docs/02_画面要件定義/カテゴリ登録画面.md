## 画面概要

新しいカテゴリを作成するための画面です。カテゴリ名、色、アイコンなどを設定できます。

## 画面機能

- カテゴリ名の入力・バリデーション
- カテゴリ色の選択
- アイコンの選択
- 重複チェック
- 入力データの保存
- 既存カテゴリの一覧表示

## 使用例とヒント

### 推奨カテゴリ設定例

| カテゴリ名 | アイコン | 色       | 説明例                 |
| ---------- | -------- | -------- | ---------------------- |
| 食費       | 🍽️        | 赤       | 食材、外食、飲み物     |
| 住居費     | 🏠        | 青       | 家賃、光熱費、通信費   |
| 交通費     | 🚗        | 緑       | 電車、バス、ガソリン   |
| 娯楽費     | 🎮        | 紫       | 映画、ゲーム、趣味     |
| 医療費     | 💊        | オレンジ | 病院、薬、健康関連     |
| 教育費     | 📚        | 茶       | 本、講座、スキルアップ |
| 衣料費     | 👕        | ピンク   | 服、靴、アクセサリー   |
| 旅行費     | ✈️        | グレー   | 旅行、宿泊、観光       |

### 入力のヒント

- **カテゴリ名**: わかりやすく短い名前にする（例：「食費」「交通費」）
- **色選択**: カテゴリの種類に応じて直感的な色を選ぶ
- **アイコン**: カテゴリの内容を表す適切なアイコンを選択
- **説明**: どのような支出が含まれるかを明記

### 注意事項

- 同じ名前のカテゴリは登録できません
- 削除済みカテゴリと同じ名前は再利用可能
- カテゴリの色とアイコンは後から変更可能
