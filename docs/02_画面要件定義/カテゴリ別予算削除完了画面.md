## 画面概要

カテゴリ別予算の削除処理が正常に完了したことをユーザーに通知する画面です。削除した予算の情報を表示し、次のアクションへの導線を提供します。

## 画面機能

- 削除完了の通知
- 削除された予算情報の表示
- 予算設定画面への遷移
- ダッシュボードへの遷移

## UI要素詳細

### 完了メッセージ

| 要素             | 内容                       | スタイル             |
| ---------------- | -------------------------- | -------------------- |
| アイコン         | ✅ チェックマーク           | 大きめのサイズ、緑色 |
| メインメッセージ | 「予算削除が完了しました」 | 大きめフォント、太字 |

### 削除結果表示

| 項目       | 表示内容              | 説明                     |
| ---------- | --------------------- | ------------------------ |
| カテゴリ名 | アイコン + カテゴリ名 | 削除されたカテゴリ       |
| 予算金額   | 削除された予算金額    | カンマ区切り表示         |
| 対象期間   | 年月                  | 予算が設定されていた期間 |

### アクションボタン

| ボタン名       | スタイル   | 動作                     |
| -------------- | ---------- | ------------------------ |
| ダッシュボード | プライマリ | ダッシュボード画面へ遷移 |
| 予算設定       | セカンダリ | 予算設定画面へ遷移       |

## 画面遷移

- **ダッシュボードボタン押下**
  - ダッシュボード画面へ遷移
  - メイン画面での予算状況確認が可能

- **予算設定ボタン押下**
  - 予算設定画面へ遷移
  - 他の予算設定や新規予算作成が可能

- **自動遷移**
  - 3 秒後に自動的にダッシュボードへ遷移（オプション）

## 表示タイミング

- **表示条件**
  - カテゴリ別の予算の削除処理が正常に完了した場合
  - データベースからの削除が成功した場合

- **表示時間**
  - ユーザーが明示的にボタンを押すまで表示
  - または 3 秒後の自動遷移（設定による）

- **エラー時の動作**
  - 削除処理が失敗した場合は、この画面は表示されない
  - エラー画面または確認画面にエラーメッセージを表示
