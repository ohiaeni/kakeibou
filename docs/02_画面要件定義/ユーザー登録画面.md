## 画面概要

新規ユーザーがアカウントを作成するための登録画面です。セキュアなユーザー認証機能を提供します。

## 画面機能

- ユーザー名の入力・バリデーション
- メールアドレスの入力・重複チェック
- パスワードの入力・強度チェック
- パスワード確認の一致チェック
- 利用規約への同意確認
- アカウント作成処理
- ログイン画面への遷移
- 入力データの保護

## UI要素詳細

### フォーム要素

| 要素名         | タイプ           | 必須 | 制約                              | 説明               |
| -------------- | ---------------- | ---- | --------------------------------- | ------------------ |
| ユーザー名     | テキスト         | ✓    | 3-20 文字、英数字とアンダースコア | ログイン時の識別子 |
| メールアドレス | メール           | ✓    | RFC準拠、重複不可                 | アカウント管理用   |
| パスワード     | パスワード       | ✓    | 8 文字以上、英数字含む            | 認証用             |
| パスワード確認 | パスワード       | ✓    | パスワードと一致                  | 入力確認           |
| 利用規約同意   | チェックボックス | ✓    | チェック必須                      | 規約への同意       |

### バリデーション

- **ユーザー名**
  - 必須入力
  - 3-20 文字
  - 英数字とアンダースコアのみ許可
  - 重複チェック（他ユーザーとの重複不可）

- **メールアドレス**
  - 必須入力
  - RFC 準拠のメール形式
  - 重複チェック（既存ユーザーとの重複不可）

- **パスワード**
  - 必須入力
  - 8 文字以上
  - 英字と数字を 1 つ以上含む
  - 特殊文字の使用可能

- **パスワード確認**
  - 必須入力
  - パスワードフィールドと完全一致

- **利用規約同意**
  - 必須チェック

## 画面遷移

- **登録ボタン押下**
  - バリデーション成功 → ログイン画面へ遷移（登録完了メッセージ表示）
  - バリデーション失敗 → 同画面でエラーメッセージ表示

- **ログインリンク押下**
  - ログイン画面へ遷移

- **規約を確認リンク押下**
  - 利用規約ページを新しいタブで開く

## セキュリティ考慮事項

- パスワードは BCrypt でハッシュ化して保存
- CSRF トークンによるフォーム保護
- 入力値のサニタイゼーション
- レート制限による登録試行の回数制限
- メールアドレス認証（オプション）
