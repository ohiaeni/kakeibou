## 画面概要

設定済みのカテゴリ別予算を削除する前に、ユーザーに削除確認を行う画面です。削除対象の予算情報を表示し、削除の意思確認を行います。

## 画面機能

- 削除対象の予算情報表示
- 削除の確認・キャンセル
- 削除実行時の画面遷移
- キャンセル時の元画面への戻り

## UI要素詳細

### 表示要素

| 要素名         | 表示内容              | 説明                     |
| -------------- | --------------------- | ------------------------ |
| カテゴリ名     | アイコン + カテゴリ名 | 削除対象のカテゴリ       |
| 予算金額       | 設定された予算金額    | 削除される予算額         |
| 対象期間       | 年月                  | 予算が設定されている期間 |
| 現在使用額     | 既に使用された金額    | 当月の支出累計           |
| 残り予算       | 予算 - 使用額         | 残りの予算金額           |
| 警告メッセージ | 削除の不可逆性の警告  | ユーザーへの注意喚起     |

### アクションボタン

| ボタン名   | スタイル         | 動作                       |
| ---------- | ---------------- | -------------------------- |
| キャンセル | セカンダリ       | 削除を中止して前画面に戻る |
| 削除       | デンジャー（赤） | 予算削除を実行             |

## 画面遷移

- **削除ボタン押下**
  - 予算削除処理を実行
  - カテゴリ別の予算削除完了の画面へ遷移

- **キャンセルボタン押下**
  - 削除処理をキャンセル
  - 予算設定画面へ戻る

- **戻るボタン押下（モバイル）**
  - 削除処理をキャンセル
  - 予算設定画面へ戻る

## データ表示

- **取得データ**
  - 削除対象の予算 ID
  - カテゴリ情報（名前、アイコン）
  - 予算金額
  - 対象年月
  - 当月の使用済み金額

- **計算項目**
  - 残り予算 = 予算金額 - 使用済み金額

- **表示形式**
  - 金額：カンマ区切り（¥50,000）
  - 期間：年/月形式（2024 年 1 月）
