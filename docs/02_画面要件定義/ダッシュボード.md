## 画面概要

ユーザーがログイン後、最初に表示されるメイン画面です。
家計簿の概要情報、最近の支出、予算状況を一覧できます。

## 画面機能

- 今月の支出サマリー表示
- 予算との比較表示
- 月次支出推移グラフ
- カテゴリ別支出円グラフ
- 最近の支出記録一覧
- クイックアクション（支出登録）
- 各詳細画面への遷移

## UI要素詳細

### サマリーカード

| カード名   | 表示内容           | データ未存在時の表示 |
| ---------- | ------------------ | -------------------- |
| 今月支出   | 当月の総支出金額   | ¥0                   |
| 今月予算   | 当月の予算総額     | 未設定               |
| 残り予算   | 予算 - 支出        | ¥0（予算未設定時）   |
| 今月登録数 | 当月の支出記録件数 | 0件                  |

### グラフ表示

- **月次支出推移グラフ**
  - 過去 12 ヶ月の支出推移を線グラフで表示
  - データなし時:「データがありません」メッセージ

- **カテゴリ別支出円グラフ**
  - 当月のカテゴリ別支出割合を円グラフで表示
  - データなし時:「データがありません」メッセージ

### 最近の支出一覧

- 最新の支出記録を 10 件表示
- 表示項目: 日付、カテゴリ、金額、説明
- データなし時:「まだ支出記録がありません」メッセージ + 支出記録ボタン

## データ取得・表示ロジック

1. **今月サマリーデータの取得**
   - 今月の支出、予算、残り予算、登録件数を計算
   - 予算が未設定の場合は「未設定」と表示

2. **グラフデータの取得**
   - 月次支出推移: 過去 12 ヶ月の月別支出データ
   - カテゴリ別支出: 当月のカテゴリ別支出集計

3. **最近の支出記録の取得**
   - 日付の降順で最新 10 件を取得
   - 支出日、カテゴリ名、金額、説明を表示

## 画面遷移

- **支出を記録するボタン** → 支出登録画面
- **予算を設定するボタン** → 予算設定画面
- **もっと見るリンク** → 支出一覧画面
- **グラフエリアクリック** → レポート画面
- **個別支出記録クリック** → 支出編集画面
