## 画面概要

新しい支出記録を入力する画面です。カテゴリ選択、金額入力、日付選択などの機能を提供します。

## 画面機能

- 新規支出記録の登録
- カテゴリ選択（ドロップダウンリスト）
- 金額入力（数値バリデーション）
- 日付選択（カレンダー機能）
- 説明入力（任意）
- レシート画像アップロード（任意）
- 入力データのバリデーション
- 最近の支出履歴表示

## UI要素詳細

### フォーム要素

| 要素名       | タイプ         | 必須 | 制約                     | 説明               |
| ------------ | -------------- | ---- | ------------------------ | ------------------ |
| カテゴリ     | ドロップダウン | ✓    | 登録済みカテゴリから選択 | 支出の分類         |
| 金額         | 数値           | ✓    | 1 円～99,999,999 円      | 支出金額           |
| 支出日       | 日付           | ✓    | 過去〜当日まで           | 支出が発生した日付 |
| 説明         | テキストエリア |      | 最大 200 文字            | 支出の詳細説明     |
| レシート画像 | ファイル       |      | JPG/PNG、最大 5MB        | レシート画像       |

### バリデーション

- **カテゴリ**
  - 必須選択
  - 登録済みカテゴリの中から選択

- **金額**
  - 必須入力
  - 数値のみ（カンマ区切り対応）
  - 1 円以上 99,999,999 円以下
  - 小数点不可

- **支出日**
  - 必須入力
  - 過去の日付〜当日まで
  - 未来日は選択不可

- **説明**
  - 任意入力
  - 最大 200 文字
  - 特殊文字制限あり

- **レシート画像**
  - 任意アップロード
  - JPG、PNG 形式のみ
  - 最大ファイルサイズ 5MB

## 画面遷移

- **保存ボタン押下**
  - バリデーション成功 → 支出一覧画面へ遷移（登録完了メッセージ表示）
  - バリデーション失敗 → 同画面でエラーメッセージ表示

- **キャンセルボタン押下**
  - 支出一覧画面へ遷移（確認ダイアログ表示）

- **戻るボタン押下（モバイル）**
  - 支出一覧画面へ遷移（確認ダイアログ表示）

## ファイルアップロード

- **対応形式**: JPEG、PNG
- **最大ファイルサイズ**: 5MB
- **アップロード方法**:
  - ファイル選択（ギャラリーから）
  - カメラ撮影（モバイルのみ）
- **プレビュー機能**: アップロード後に画像プレビュー表示
- **削除機能**: アップロード済み画像の削除可能
