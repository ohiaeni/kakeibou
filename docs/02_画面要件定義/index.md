### Thymeleaf MPA構成の特徴
- **サーバーサイドレンダリング**: テンプレートはサーバー側で処理され、完全なHTMLとしてクライアントに送信
- **Spring Boot統合**: コントローラーからModelを通じてデータをテンプレートに渡す
- **テンプレートフラグメント**: 共通部品（ヘッダー、フッター、ナビゲーション）の再利用
- **フォーム処理**: Thymeleafの標準的なフォームバインディングとバリデーション
- **セッション管理**: Spring Securityとの統合によるセッション管理
- **国際化対応**: Thymeleafのメッセージ機能による多言語対応
- **CSRFトークン**: Spring Securityによる自動CSRF保護

- **エラー**: #F44336 (Red)
