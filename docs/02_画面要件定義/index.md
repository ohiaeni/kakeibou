### Thymeleaf MPA構成の特徴

- **サーバーサイドレンダリング**: テンプレートはサーバー側で処理され、完全な HTML としてクライアントに送信
- **Spring Boot統合**: コントローラーから Model を通じてデータをテンプレートに渡す
- **テンプレートフラグメント**: 共通部品（ヘッダー、フッター、ナビゲーション）の再利用
- **フォーム処理**: Thymeleaf の標準的なフォームバインディングとバリデーション
- **セッション管理**: Spring Security との統合によるセッション管理
- **国際化対応**: Thymeleaf のメッセージ機能による多言語対応
- **CSRFトークン**: Spring Security による自動 CSRF 保護

- **モバイル**: 767px 以下

### カラーパレット

- **プライマリ**: #1976D2 (Blue)
- **セカンダリ**: #424242 (Grey)
- **アクセント**: #FF5722 (Deep Orange)
- **エラー**: #F44336 (Red)
- **警告**: #FF9800 (Orange)
- **成功**: #4CAF50 (Green)

### フォント

- **メイン**: 'Noto Sans JP', sans-serif
- **数字**: 'Roboto Mono', monospace
