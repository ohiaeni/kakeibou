## 画面概要

ユーザーがアプリケーションにログインするための認証画面です。

## 画面レイアウト

### デスクトップ版
```
┌─────────────────────────────────────┐
│              Header                 │
│         家計簿アプリ ロゴ            │
└─────────────────────────────────────┘
│                                     │
│        ┌─────────────────┐          │
│        │                 │          │
│        │   ログインフォーム   │          │
│        │                 │          │
│        │  [メールアドレス]   │          │
│        │  [パスワード]      │          │
│        │                 │          │
│        │   [ログイン]      │          │
│        │                 │          │
│        │ アカウントをお持ちで │          │
│        │ ない方は          │          │
│        │   [新規登録]      │          │
│        └─────────────────┘          │
│                                     │
└─────────────────────────────────────┘
```
