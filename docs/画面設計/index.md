# 画面設計

このドキュメントでは、家計簿アプリ（Vue.js）の画面設計に関する情報を提供します。各画面のレイアウト、コンポーネント、ユーザーインターフェースの要素について詳しく説明します。

## 画面構成概要

### フロントエンド技術スタック
- **フレームワーク**: Vue.js 3
- **UI ライブラリ**: Vuetify 3 / Element Plus
- **状態管理**: Pinia
- **ルーティング**: Vue Router 4
- **HTTP クライアント**: Axios
- **ビルドツール**: Vite

### 画面一覧
1. [ログイン画面](./ログイン画面.md) - ユーザー認証
2. [ユーザー登録画面](./ユーザー登録画面.md) - 新規ユーザー登録
3. [ダッシュボード](./ダッシュボード.md) - メイン画面、支出概要表示
4. [支出記録画面](./支出記録画面.md) - 支出の入力・編集
5. [支出履歴画面](./支出履歴画面.md) - 支出記録の一覧表示・検索
6. [予算設定画面](./予算設定画面.md) - 月次予算の設定・管理
7. [レポート画面](./レポート画面.md) - 支出分析・グラフ表示
8. [設定画面](./設定画面.md) - ユーザー設定・カテゴリ管理

## 共通設計原則

### レスポンシブデザイン
- **デスクトップ**: 1200px以上
- **タブレット**: 768px - 1199px
- **モバイル**: 767px以下

### カラーパレット
- **プライマリ**: #1976D2 (Blue)
- **セカンダリ**: #424242 (Grey)
- **アクセント**: #FF5722 (Deep Orange)
- **エラー**: #F44336 (Red)
- **警告**: #FF9800 (Orange)
- **成功**: #4CAF50 (Green)

### フォント
- **メイン**: 'Noto Sans JP', sans-serif
- **数字**: 'Roboto Mono', monospace
